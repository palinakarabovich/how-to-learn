## Описание проекта
Данный проект представляет собой одностраничный лендинг о приемах эффективного обучения. Учебная работа при прохождени курса Веб Разработки в Яндекс.Практикуме.

## Инструменты и технологии
* HTML5,  
* CSS3
* flexbox,
* позиционирование,
* БЭМ,
* анимация,
* YouTube Player API
